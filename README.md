# SIWS PHP Template

This is a lightweight PHP + Bootstrap template created for a student project to resemble the look-and-feel of a college website. Replace images and text with your own content. Run using XAMPP or any PHP server.

## How to run
1. Place the project folder inside your webserver's document root (e.g., `htdocs` for XAMPP).
2. Start Apache & MySQL if needed.
3. Open `http://localhost/siws-php-clone/` in your browser.

## Customization
- Replace images in `assets/images/` with real photos (logo.png, hero.jpg, amenity1.jpg ...).
- Edit content in the PHP files.
- Add dynamic content by connecting to a database using `mysqli`/`PDO`.
